# Guardies-IA

Sistema de gestió d'incidències per a equips de guàrdia tècnica.

## 📋 Descripció

Aquest projecte permet registrar, gestionar i fer seguiment d'incidències tècniques de manera senzilla, amb emmagatzematge de fotos, historial de notes, gestió de proveïdors i exportació de dades.

Està desenvolupat amb el framework **IA-Productiva**, que permet que una IA (DeepSeek, ChatGPT, Claude...) col·labori en el desenvolupament de nous projectes de forma estructurada.

## 🚀 Funcionalitats (MVP)

- [x] Plantilla de registre d'incidències
- [x] Formulari per introduir incidències
- [x] Emmagatzematge de fotos
- [x] Gestió de contactes de proveïdors
- [x] Historial d'incidències
- [x] Generació d'informes
- [x] Interfície per pujar i veure fotos
- [x] Sistema de comentaris i notes
- [x] Exportació a CSV i PDF
